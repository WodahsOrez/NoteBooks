# 主标题

## 数据类型

![1550393917847](images/grammar/1550393917847.png)

| 类型    | 范围                    | 备注    |
| ------- | ----------------------- | ------- |
| byte    | 8位(-2^7^~ 2^7^-1)      |         |
| short   | 16位(-2^15^~2^15^-1)    |         |
| int     | 32位(-2^31^~2^31^-1)    |         |
| long    | 64位(-2^63^~2^63^-1)    | 末尾加L |
| float   | 32位(1符号8指数23尾数)  | 末尾加f |
| double  | 64位(1符号11指数52尾数) |         |
| char    | 16位(0~2^16^-1)         |         |
| boolean | true/false              |         |
